# Alminana Eiler B. Portfolio

Personal portfolio website for Alminana Eiler B., a Web Developer based in Al Qassim, Saudi Arabia.

## Focus

- Laravel, PHP, React.js, and Node.js web development
- E-commerce, admin tools, APIs, and database-driven systems
- Website maintenance, debugging, deployment support, and technical support

## Tech Stack

- Next.js
- React
- Tailwind CSS
- PrimeReact

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
